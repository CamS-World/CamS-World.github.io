Island Adventure

MVC Website

This MVC website was designed to allow for visitors of the site to view possible vacation spots among island chains. We were tasked with organizing these vacation regions by oceans, island chains, countries, and continents.

To complete this project my team and I utilized scrum and agile methodologies, GitHub for source control, Spring MVC to develop our templates in Java and HTML, CSS, and thymeleaf for our front end.
