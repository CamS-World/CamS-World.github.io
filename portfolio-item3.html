accessAbilities

Single Page Application

The accessAbilities app is designed to assist disabled persons in their daily lives ensuring that access is never limited. The app features a list of business that can be filtered by specific accessability features, maps, the ability for users to leave reviews, and resources to help find employment.

To complete this project my team and I utilized scrum and agile methodologies, GitHub for source control, Java for our custom backend and HTML, CSS, and JavaScript for our front end. We utilized Rest API as well.
